# 示例

## 输入（低质量题面片段）

```text
题目描述
给定n个整数a1,a2...an,请你求出最大值和最小值之差.

输入格式
第一行一个整数n
第二行n个整数

输出格式
输出一个整数表示答案.

数据范围与提示
1<=n<=2e5, -1e9<=ai<=1e9

样例输入
5
1 3 2 9 4

样例输出
8
```
